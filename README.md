## 🐶 프로젝트 소개
<img src='https://i.postimg.cc/sXDfWYbv/image.gif' border='0' alt='image'/>
Spring Boot로 만든 반려동물 용품 사이트입니다.
<br/>
- '산책가자'는 반려동물이 가장 좋아하는 소리로 많은 반려동물 가족들을 위한 좋은 제품들을 제공합니다.
<br/>


## ⏲ 개발 기간
* 2022.11.01 - 2023.03.17
<br>

## 🙇‍♀️ 멤버 구성
* 정선주 (팀장)
    * 메인 페이지, 상품 카테고리 페이지
  * DB 설계 및 UI 구현
* 유미현
    * 상품 등록 (이미지 등록)
    * 상품 상세 페이지 - 카카오톡 공유 API 사용
    * 장바구니, 위시리스트
    * 주문 및 결제 페이지 - 결제 API 사용
    * 상품 Q&A 게시판
  * DB 설계 및 UI 구현
  * PPT 제작
* 최병현
    * 리뷰, FAQ, 공지사항, Q&A
  * DB 설계 및 UI 구현
* 김민성
    * 로그인 / 회원가입
    * 마이페이지 회원 정보, 배송지
    * 쿠폰, 주문내역 조회
  * DB 설계 및 UI 구현
