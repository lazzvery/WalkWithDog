## 🐶 프로젝트 소개

Spring Boot로 만든 반려동물 용품 사이트입니다.
<br/>
- '산책가자'는 반려동물이 가장 좋아하는 소리로 많은 반려동물 가족들을 위한 좋은 제품들을 제공합니다.
<br/>
<br/>


## ⏲ 개발 기간
* 22.11.01일 - 23.03.17일
<br>

## 🧑‍🤝‍🧑 멤버 구성
 - 정선주 - 메인 페이지, 상품 카테고리 페이지, 발표
 - 유미현 - 상품 상세 페이지, 장바구니, 위시리스트, 주문/결제,
           상품 Q&A 게시판, 상품(이미지) 등록, PPT 제작, 발표
 - 최병현 - 리뷰 게시판, FAQ 게시판, 공지사항 게시판, Q&A 게시판, 발표
 - 김민성 - 로그인/회원가입, 마이페이지, 관리자 페이지, 발표
<br>

## 📚 개발 환경 / STACKS
- `Java 8`
- `JDK 1.8.0`
- **Framework** : Springboot(2.7.10)
- **Database** : MySQL
- **ORM** : Mybatis
