## 🐶 프로젝트 소개
<img src="https://i.postimg.cc/QCfs9KD2/1-logo.png" width="25%">
Spring Boot로 만든 반려동물 용품 사이트입니다.
<br/>
- '산책가자'는 반려동물이 가장 좋아하는 소리로 많은 반려동물 가족들을 위한 좋은 제품들을 제공합니다.
<br/>
<br/>


## ⏲ 개발 기간
* 22.11.01일 - 23.03.17일
<br>

## 🙇‍♀️ 멤버 구성
* 정선주
    * 메인 페이지, 상품 카테고리 페이지
    * 상품 정렬, 상품 검색
  * DB 설계 및 UI 구현
* 유미현
    * 상품 및 이미지 등록(추가)
    * 상품 상세 페이지(조회) - 카카오톡 공유 API 사용
    * 장바구니, 위시리스트 (조회/추가/수정/삭제)
    * 주문 및 결제 페이지 - 결제 API 사용 (추가)
    * 상품 Q&A 게시판 (조회/추가/수정/삭제)
  * DB 설계 및 UI 구현
  * PPT 제작
* 최병현
    * 리뷰 게시판, FAQ 게시판, 공지사항 게시판, Q&A 게시판 (조회/추가/수정/삭제)
    * 게시글 정렬, 게시글 비밀번호 등록 및 확인
  * DB 설계 및 UI 구현
* 김민성
    * 로그인/회원가입 페이지
    * 아이디 찾기 및 비밀번호 찾기
    * 회원정보 변경(수정), 배송지(추가/수정/삭제/조회)
    * 쿠폰(조회), 주문내역 및 상세내역(조회)
    * 회원(조회), 상품(조회), 커뮤니티 게시글(조회)
  * DB 설계 및 UI 구현
<br>
